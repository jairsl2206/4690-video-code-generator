La configuración se guarda automáticamente en tu navegador.

Exporta tu configuración para compartirla o respaldarla. Importa archivos .json para cargar otras configuraciones.

Parámetros del JSON de configuración:

  • simulation.type — "all", "editor", "powershell", "ibm4690"
  • simulation.fontSize — Tamaño de fuente (0.6-1.5 rem)
  • simulation.videoFormat — "mp4" o "webm"
  • simulation.videoPrefix — Prefijo para archivos (ej. "1.1-tipos")
  • simulation.speedTyping — ms/carácter en editor (10-150)
  • simulation.speedTerminal — ms/carácter en terminal (10-150)
  • simulation.speedOutput — ms/línea de salida (50-800)
  • simulation.pauseBetweenPhases — ms entre paneles (200-3000)
  • editor.fileName — Nombre del archivo .BAS
  • editor.code — Código fuente BASIC
  • powershell.prompt — Texto del prompt
  • powershell.dirTemplate — Plantilla del comando DIR
  • powershell.steps[] — Array de pasos configurables
  • powershell.steps[].enabled — true/false para habilitar paso
  • powershell.steps[].command — Comando a mostrar
  • powershell.steps[].output — Salida del comando (usa {{PS_DIR_OBJ}} o {{PS_DIR_EXE}})
  • powershell.steps[].simulatedDelay — ms de espera simulada
  • powershell.steps[].outputDelay — ms entre líneas de salida
  • ibm4690.prompt — Prompt del controlador
  • ibm4690.bootSequence — Texto de arranque del sistema
  • ibm4690.dirTemplate — Plantilla DIR del 4690 OS
  • ibm4690.steps[] — Array de pasos configurables
  • ibm4690.steps[].enabled — true/false
  • ibm4690.steps[].command — Comando a ejecutar
  • ibm4690.steps[].output — Salida del comando (usa {{DOS_DIR_RUN}})
  • ibm4690.steps[].simulatedDelay — ms de espera simulada
  • ibm4690.steps[].outputDelay — ms entre líneas de salida
